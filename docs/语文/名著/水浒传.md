# 水浒传
## 主要人物
### 鲁达 (天孤星)
- 绰号：花和尚
- 性格：**粗中有细**，**豁达明理**，爱憎分明/嫉恶如仇，侠肝义胆，勇而有谋，扶危济困
- 主要情节：
    - 拳打镇关西
    - 大闹五台山
    - 大闹桃花村
    - 火烧瓦罐寺
    - 倒拔垂杨柳
    - 大闹野猪林
    - 单打二龙山
    - 斗呼延灼
    - 擒夏侯成
    - 捉方腊
    - 浙江六和寺圆寂

### 林冲 (天雄星)
- 绰号:豹子头
- 性格：逆来顺受/妥协忍让，**委屈求全**，安分守己，循规蹈矩 到 **敢于反抗**并且反抗彻底，勇而有谋，武艺高强
- 主要情节：
    - 误入白虎堂
    - 刺配沧州道
    - 棒打洪教头
    - 风雪山神庙
    - 雪夜上梁山
    - 火并王伦
    - 中风在六和寺养病，武松照顾他，半年后去世

### 杨志 (天暗星)
- 绰号：青面兽
- 性格：看重功名/**急功近利**，勇猛刚烈，精明谨慎，粗暴蛮横
- 主要情节：
    - 失陷花石纲
    - 与林冲斗武
    - 杨志卖刀(杀牛二)
    - 北京斗武
    - 押送生辰纲
    - 双夺宝珠寺
    - 征方腊时途中病故

### 武松 (天伤星)
是下层英雄好汉中最富有血性和传奇色彩的人物
- 绰号：行者
- 性格：性格刚烈，勇而有谋，崇尚忠义/**重情重义**，有恩必报有仇必复
- 主要情节：
    - 景阳冈打虎
    - 血刃潘金莲、斗杀西门庆
    - 十字坡打店
    - 醉打蒋门神
    - 大闹飞云浦
    - 血溅鸳鸯楼
    - 蜈蚣岭除恶
    - 醉打孔亮
    - 征方腊途中被包道乙暗算失去右臂，后在六和寺出家，至八十善终

### 吴用 (天机星)
- 绰号：智多星
- 性格：足智多谋/神机妙算，沉着冷静，善于用人，深谋远虑
- 主要情节：
    - 说三阮撞筹
    - 智取生辰纲
    - 智激林冲火并王伦
    - 梁山伯举戴宗
    - 双用连环计助力破祝家庄
    - 使时迁盗甲
    - 赚金玲吊桂(救了史进、鲁智深)
    - 智赚卢俊义
    - 智取大名府
    - 用十面埋伏计大破童贯
    - 智取文安县
    - 征方腊后受封为官，最后与花荣在宋江墓前自缢而死

### 宋江 (天魁星)
- 绰号：及时雨
- 性格：**精明练达**，仗义疏财，扶危济困，忠君孝顺，胸怀宽广，善于用人，有组织军事才能，但 **虚假伪善**
- 主要情节：
    - 私放晁盖
    - 怒杀阎婆惜
    - 宋江夜看小鳌山
    - 揭阳岭逢李俊
    - 及时雨会神行太保
    - 浔阳楼题反诗
    - 智取无为军
    - 遇九天玄女
    - 三打祝家庄
    - 大破连环马
    - 闹西岳华山
    - 兵打北京城
    - 夜打曾头市
    - 三败高太尉
    - 被毒酒毒死

### 李逵 (天杀星)
- 绰号：黑旋风
- 性格：**直爽率真**，嫉恶如仇，侠肝义胆，忠义孝顺，勇猛无畏，脾气火爆，**头脑简单**，**滥杀无辜**
- 主要情节：
    - 黑旋风斗浪里白跳
    - 李逵单身劫法场
    - 真假李逵 / 假李逵剪径劫单人
    - 沂岭杀四虎
    - 李逵打死殷天锡
    - 李逵斧劈罗真人
    - 黑旋风探穴救柴进
    - 李逵元宵闹东京
    - 黑旋风乔捉鬼
    - 黑旋风扯诏骂钦差
    - 被宋江用毒酒毒死
