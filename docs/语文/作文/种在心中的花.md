# 种在心中的花
&emsp;&emsp;树影斑驳地铺满小径，阳光洒下一地的温情和烂灿。我走在落英遍布的小径上，闭上双眼，“咕嘟”的声音萦绕耳畔，一朵花早已在心中种下。

&emsp;&emsp;幼时的我体弱多病，每每来到祖母家，她总亲手熬制各种药让我眼下清晨，一乘微光透过斑驳的树隙，温柔地把我唤醒。今风朦胧了我的双眼，迷离中我走进厨房。熟悉的药味将我唤醒，额前有些晕乎乎的我，熟练地趴在桌上。

&emsp;&emsp;迷迷乎乎间，我看到祖母瘦小的背影在空旷的厨房里显得格外突兀。她佝使着背，在暖色迷离的月光下，把金银花、胖大海等物放进药罐里，将贝母粉和水倒入。她用布干擦拭着瓦罐。开火，点炉。她将互罐放置在炉上，蓬红相间的光火跳琢着，柔柔地舔着拭着罐底。她用围裙抹了一把手，拿出调羹轻轻地搅拌几下，涩涩的药香慢慢地弥漫了房间。她将耳盖盖上，搬了一个小板凳坐在炉子旁，手持一把蒲扇，缓缓的地扇着。一缕一缕的袅袅白雾，一点一点地围笼住她，一丝丝地朦胧了这片样和的天地。听话的从罐口落流碗中。祖母用布包裹碗底，将勺子放进碗中，不停搅拌地着，直到不那么烫了才端给我，怜惜地劝:“囡囡快喝吧，喝了就不难受了。”

&emsp;&emsp;我将碗轻轻捧起。白色的碗包裹的着深褐的汁。轻吹一口，深褐的药汁上荡漾着圈圈波纹。轻呡一口，唇齿间尽是草药的芬芳。闭眼，一股脑地闷随着时间的推移，起初的苦涩已散去，只有淡淡的药香。

&emsp;&emsp;我抬头望向祖母，咧开一个笑容。向她夸药的好。袅袅白雾染色了她的青丝。根根银发，像冬雪落地，胜过秋日的霜。她青筋暴起的双手握在一起。在听到我的夸奖后，布满皱纹的脸上刹时扬起一个笑容。那如往事云烟的纹似片片花瓣，聚抚成美丽的花儿，深深地种在我心中。

&emsp;&emsp;蒸气如烟雾般弥漫了这一片天地，一老一少的笑声消散于这朦胧的烟中，一片温情。

&emsp;&emsp;“嘎吱”，我睁开双眼，推开门。“囡囡来了”那熟悉的花又绽放。我久久地凝视着那花，与童年种在心中的花一模一样。

&emsp;&emsp;春风轻拂，院中的花纷纷飘落，滑落在心中的那一朵花上。一整个春天的烂漫却随着温情的花种在心中。
