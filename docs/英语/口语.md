# 口语
- That's right 你说的是正确的
- All right 回答建议
- That's all right 回答道歉 / 回答感谢
<br/>
- That's true 用于同意一个陈述
- It's true 在陈述一个事实
<br/>
- Never mind 没关系 / 不用了
- No wonder 难怪
- No worries 别担心
- Not at all 回答感谢 / 客套话 / 回答道歉
<br/>
- You've made my day 你今天让我真高兴 / 你成就了如今的我 / 你使我的生活特别有意义
- You've got this 你能行
- All the best 一切顺利
- It depends 看情况
<br/>
- You('ve) got me there 你把我问住了
- Take your time 别着急
- That's the case 确实如此 / 用于推测后得出结论 / 表示情况依然保持不变
- That's not the case 事实并非如此
<br/>
- ask for the moon 某个人的要求很过分，想要得不到的东西

## 回答道歉
- Never mind

- Not at all
- That's all right

- My pleasure
