# 第二课 构建人类命运共同体
## 必背知识点:
1. 当今世界经济全球化，政治多极化，文化多样化。经济全球化趋势不可逆转。世界格局向多极化发展。
2. 当今世界，人们越来越致力于以和平谈判代替武力解决争端。
3. 当今人类必须共同面对的发展课题是：消除贫困。
4. 各国解决全球性问题的必然选择：构建人类命运共同体。
5. 全人类共同价值：和平、发展、公平、正义、自由、民主
6. 我国坚持通过对话协商，以和平手段解决国际争端和热点难点问题，永远做维护世界和平的坚定力量。

### 维护和平，促进发展的途径：
1. 各国角度：
    1. 致力于以和平谈判代替武力解决争端。
    2. 结合自身的国情，不断探索消除贫困的有效途径，推动可持续发展。
2. 我国角度：
    1. 通过对话协商，以和平手段解决国际争端，做维护世界和平的坚定力量。
    2. 脱贫攻坚战取得全面胜利，为世界减贫事业和发展贡献了中国力量。
3. 公民角度：
    1. 珍惜和平，主动承担维护世界和平的责任。
    2. 关注世界的发展，关注人类的命运。

### 为什么要构建人类命运共同体?
1. 当今世界，各国相互联系、相互依存的程度空前加深。
2. 人类面临许多共同挑战，需要解决许多全球性问题。
3. 采取共同行动，承担共同责任，构建人类命运共同体，应成为各国解决全球性问题的必然选择。
4. 没有国家能独自面对各种挑战，也不能退回到自我封闭的孤岛。
5. 有利于加强合作，维护各国人民利益，促进世界和平与发展。

### 各国构建人类命运共同体?
各国要努力扩大利益的交汇点，谋求开放创新、包容互惠的发展前景。
1. 坚持对话协商，建设一个持久和平的世界
2. 坚持共建共享，建设一个普遍安全的世界；
3. 坚持合作共赢，建设一个共同繁荣的世界；
4. 坚持交流互鉴，建设一个开放包容的世界；
5. 坚持绿色低碳，建设一个清洁美丽的世界。


### 中国为构建人类命运共同体 (应对全球性问题) 做出了哪些重大贡献?
中国积极开展大国特色外交，推动建设新型国际关系，推动构建人类命运共同体，弘扬和平发展、公平正义、民主自由的全人类共同价值，引领人类进步潮流。<br/>
 (实例：脱贫攻坚战胜利；一带一路建设；积极参与全球治理；扩大对外开放；新冠疫情时派驻医护人员、提供物资) 

### 各国人民、公民、青少年、能为构建人类命运共同体做些什么?
1. 相互信任、守望相助和共同担当。
2. 关怀生命、尊重生命的价值。
3. 把构建人类命运共同体的理念同实际行动联系起来。
4. 既心系祖国，又关注世界发展，关注人类命运
5. 在为人民利益的不懈奋斗中书写人生华章。

## 熟读知识点:
1. 国际社会为维护世界和平采取的措施：成立联合国、派遣维和部队、签署核不扩散条约
2. 威胁世界和平的因素有：
局部战争与冲突从未间断，霸权主义、强权政治、民族问题、宗教冲突、领土争端和恐怖主义等。
3. 世界经济现状：世界经济发展不平衡的现象仍然存在，发达国家和发展中国家之间的差距仍然很大。

### 人类面临的世界性问题有哪些?
1. 世界面临的不稳定性不确定性突出
2. 世界经济增长动力不足
3. 贫富分化日益严重
4. 地区热点问题此起彼伏
5. 恐怖主义、网络安全、重大传染性疾病、气候变化等非传统安全威胁持续蔓延。
