# 九下第三课 与世界紧相连
## 必背知识点:
1. 现阶段我国的基本国情是：我国仍处于并将长期处于**社会主义初级阶段**。<br/>
这需要我们：全面深化改革，统筹国际国内两个大局，走和平发展道路，奉行互利共赢的开放战略。
2. 中国的全球治理观：以**共商共建共享**为原则的全球治理观。
3. 中国的国际形象：**事不避难，勇于担当**。中国始终是**世界和平的建设者、全球发展的贡献者、国际秩序的维护者**，已成为**更具国际影响力、创新引领力、道义感召力**的**负责任大国**。
4. 对待人类文明成果的态度：要学习和借鉴人类文明的一切优秀成果，坚持以我为主，兼收并蓄。

### 中国面对各种区域性和全球性的难题与危机承担了哪些责任? (我国对世界的贡献和影响?)  (中国怎样与世界深度互动?)  (中国担当表现在哪些方面?) 
1. 中国积极参与引领全球治理体系改革和建设，主动承担国际责任，既尽力而为，又量力而行。
2. 中国着眼时代发展大势，遵循共商共建共享原则，为全球治理提出中国方案，贡献中国智慧。
3. 中国广泛参与国际事务，发挥负责任大国作用，促进人类社会共同发展。
4. 中国推动国际秩序朝着更加公正合理的方向发展。
5. 中国始终是世界和平的建设者、全球发展的贡献者、国际秩序的维护者。
6. 中国文化对世界的影响越来越大，让世界感受中国文化的魅力。
7. 中国正为世界经济增长注入新活力，日益成为世界经济发展的引擎和稳定器。
8. 中国是世界格局中的重要力量，推动构建人类命运共同体，建设一带一路，使世界向着公平公正、多元共治、包容有序的格局发展。

## 熟读知识点
### 中华文明为什么需要与其他文明交流互鉴?世界文明为什么需要交流互鉴?
1. 中华文明在交流互鉴中发展。
2. 中国从不同文明中寻求智慧、汲取营养，不仅有助于自身文明的发展，而且能推动世界文明的进步。
3. 文明因交流而多彩，文明因互鉴而丰富。
4. 世界各国应以文明交流超越文明隔阂、文明互鉴超越文明冲突、文明共存超越文明优越，共同应对各种全球性挑战。
