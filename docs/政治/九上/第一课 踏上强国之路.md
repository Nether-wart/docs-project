# 第一课 踏上强国之路
## 必背知识点:
1. 中国共产党的初心和使命：**为中国人民谋幸福，为中华民族谋复兴**。
2. 党和政府坚持的发展思想：**以人民为中心的发展思想**。
3. 党的奋斗目标：**人民对美好生活的向往**。 (注意与“党的最高理想和最终目标”的区别) 
4. 社会主义初级阶段所有工作的中心是 **以经济建设为中心**。 (一个中心) 
5. 1978年党的十一届三中全会召开，开启了**改革开放**的历史征程。
6. 发展的根本目的就是 **增进民生福祉**。
7. 我国经济发展已由 **高速增长阶段转向高质量发展阶段**。 (经济发展新常态) 
8. 全面建设社会主义现代化国家的首要任务是 **高质量发展**。
 (实现高质量发展，需要转变发展方式，优化经济结构，转换增长动力，建设现代化经济体系) 
9. 社会主义的本质要求是 **解放和发展社会生产力**。
10. **共同富裕**是中国特色社会主义的本质要求，也是一个长期的历史过程。

### 新时代我国社会主要矛盾是什么?解决这一矛盾主要靠什么?
1. 社会主要矛盾是：**人民日益增长的美好生活需要和不平衡不充分的发展之间的矛盾**。
2. 解决这一矛盾主要靠 **开启全面深化改革的新征程**。

### 改革开放的重要性?
1. 改革开放是我国的强国之路。
2. 改革开放是当代中国最鲜明的特色。
3. 改革开放是决定当代中国命运的关键一招 (关键抉择) ，也是决定实现中华民族伟大复兴的关键一招。

### 为什么要全面深化改革? (为什么改革只有进行时，没有完成时?) 
1. 进入新时代 (中共十八大) ，我国社会主要矛盾已经转化为人民日益增长的美好生活需要和不平衡不充分的发展之间的矛盾，需要开启全面深化改革的新征程。
2. 我国经济已由高速增长阶段转向高质量发展阶段，需要转变发展方式，建设现代化经济体系。
3. 我国经济发展还面临区域发展和城乡发展不平衡不协调等现实挑战。
4. 改革开放是当代中国最鲜明的特色。是决定中国命运的关键一招 (关键抉择) 。
5. 改革开放是强国之路、富民之路。
6. 改革开放极大解放和发展了社会生产力。

### 为什么要共享发展成果? (或为什么重视民生问题?) 
1. 共同富裕是中国特色社会主义的本质要求，也是一个长期的历史过程。
2. 人民对美好生活的向往，就是党的奋斗目标。
3. 发展的根本目的就是增进民生福祉。

### 怎样共享发展成果?(为了共享发展成果，党和政府采取了哪些措施?)
1. 坚持以人民为中心的发展思想，人人参与尽力、人人享有，向共同富裕前进。
2. 抓住人民最关心最现实的利益问题，不断满足人民日益增长的美好生活需要。
3. 在发展中补齐民生短板，促进社会公平正义。
- 党和政府具体做了以下实事：
坚决打赢脱贫攻坚战；完善社会保障体系；实施健康中国战略；提高就业质量和人民收入水平

## 熟读知识点:
1. **强国富民**成为一百多年来中华民族矢志不渝的奋斗目标。
2. 改革开放形成的社会共识：**尊重劳动、尊重知识、尊重人才、尊重创造**。
3. 中国已经成为世界第二大经济体。

### 中国共产党是如何带领中国人民走上强国富民之路的?
1. 1949年，完成新民主主义革命，建立中华人民共和国，中华民族站起来了。
2. 1956年完成社会主义革命，确立社会主义基本制度，为中国发展奠定坚实的基础。
3. 1978年，进行改革开放新的伟大革命，中华民族开始富起来。
4. 十八大以来，中国特色社会主义进入新时代，中国共产党带领人民走上强国之路，中华民族迎来从站起、富起来到强起来的伟大飞跃。

### 改革开放以来，取得一切成就和进步的原因? (中国腾飞的原因?) 
1. 中国共产党领导中国人民开辟了中国特色社会主义道路，形成了中国特色社会主义理论体系，确立了中国特色社会主义制度，发展了中国特色社会主义文化。 (根本原因) 
2. 共产党的领导和中国人民团结奋斗，国家富强、民族振兴让中国人更加自信。
3. 坚持改革开放、共享发展成果。
4. 坚持科教兴国、人才强国、创新驱动发展战略。
5. 坚持节约资源、保护环境的基本国策，
6. 贯彻创新、协调、绿色、开放、共享的新发展理念。
7. 坚定文化自信，弘扬中国精神，凝聚中国力量。
8. 统筹“五位一体”总体布局、协调“四个全面”战略布局。
