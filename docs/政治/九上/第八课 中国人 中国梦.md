# 第八课 中国人 中国梦
## 必背知识点:
1. 近代以来中华民族最伟大的梦想 (中国梦) ：国家富强、民族振兴、人民幸福就是要实现中华民族的伟大复兴。
2. 新时代：十八大以来 (2012年) ，中国特色社会主义进入了新时代。
新征程：迈上全面建设社会主义现代化国家的新征程。
4. 当代马克思主义、二十一世纪马克思主义：习近平新时代中国特色社会主义思想。
5. “四个自信”：道路自信、理论自信、制度自信、文化自信。
6. 中国精神：以爱国主义为核心的民族精神和以改革创新为核心的时代精神。
7. 中国道路：中国特色社会主义道路。
8. 中国力量：全国各族人民大团结的力量。

### 进入新时代，中国特色社会主义伟大事业的指导思想
1. 马克思列宁主义
2. 毛泽东思想
3. 邓小平理论
4. “三个代表”重要思想
5. 科学发展观
6. 习近平新时代中国特色社会主义思想
-  ( 3、4、5、6为中国特色社会主义理论体系 ) 

### 怎样实现中国梦? (国家角度) 
1. 坚持党的领导
2. 走中国道路
3. 弘扬中国精神 
4. 凝聚中国力量

### 青少年怎样为实现中国梦、民族梦做奉献? (公民、个人角度) 
1. 树立远大理想，认真学习，注重实践，培养创新精神，提高创新能力。
2. 发扬艰苦奋斗精神，努力承担社会责任，积极参与公益活动。
3. 热爱祖国，坚定文化自信，弘扬中华传统美德，中华民族精神和社会主义核心价值观。
4. 关心国家发展，自觉维护国家安全、荣誉和利益，维护祖国统一和民族团结。
5. 树立节约资源，保护环境的理念，从小事做起，践行绿色低碳生活方式。

### “两个一百年”奋斗目标
- 到建党一百年时 (1921-2020年) ，**全面建成小康社会** (2020年已经实现) ；
- 到新中国成立一百年时 (1949-本世纪中叶) ，**全面建成社会主义现代化强国** (正在迈进) 。

### 全面建设社会主义现代化强国，分哪两步走?
1. 从2020到2035年，在全面建成小康社会的基础上，**基本实现社会主义现代化**。
2. 从2035到21世纪中叶，在基本实现现代化的基础上，**把我国建设成为富强民主文明和谐美丽的社会主义现代化强国**。

### 中国自信、民族自信的根本所在是什么? (为什么中国人越来越自信?) 
1. 中国自信、民族自信的根本所在：
    1. 开辟了中国特色社会主义道路；
    2. 形成了中国特色社会主义理论体系；
    3. 确立了中国特色社会主义制度；
    4. 发展了中国特色社会主义文化。
2. 在党的领导下，中国特色社会主义伟大事业不断取得新成就，国家富强、民族振兴让中国人更加自信。

### 自信的中国人有哪些表现?
1. 对国家有认同 
2. 对文化有底气 
3. 对发展有信心。
4.  (心态上) 培育理性平和、不卑不亢、开放包容的心态；
5.  (行动上) 既要胸怀理想又要求真务实，既要满怀激情又要锲而不舍；
6. 不忘初心，坚持“四个自信”；
7. 发扬艰苦奋斗的精神，彰显自信品格，与时代同进步。

## 熟读知识点:
1. 中国梦，体现了中华民族和中国人民的整体利益，是国家的梦、民族的梦也是每个中国人的梦。
2. 中国共产党团结带领中国人民进行的一切奋斗、一切牺牲、一切创造，归结起来就是一个主题：实现中华民族的伟大复兴。
3. 办好中国的事情，关键在党。
4. 伟大建党精神：弘扬坚持真理、坚守理想，践行初心、担当使命，不怕牺牲、英勇斗争，对党忠诚、不负人民的伟大建党精神。
5. “五位一体”总体布局：经济建设、政治建设、文化建设、社会建设、生态文明建设。
6. “四个全面”战略布局：全面建设社会主义现代化国家、全面深化改革、全面依法治国、全面从严治党。
