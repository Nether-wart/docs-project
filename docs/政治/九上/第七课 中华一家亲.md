# 第七课 中华一家亲
## 必背知识点:
1. 我国处理民族关系的方针：**民族平等、民族团结和各民族共同繁荣**。
2. 我国处理民族关系的基本政治制度：**民族区域自治制度**。
3. 社会主义民族关系：**平等、团结、互助、和谐**。
4. 中华民族的最高利益：**加强和巩固民族团结，维护国家统一**。
5. “同心掬得满庭芳”，各族人民要铸牢：**中华民族共同体意识**。
6. 解决台湾问题的基本方针，实现国家统一的最佳方式是：**“和平统一、一国两制”**。
7. 两岸关系的政治基础是一个中国原则，必须坚持“九二共识”，坚决反对“台独”。

### 保持香港、澳门长期繁荣稳定，
1. 必须贯彻“一国两制”、“港人治港”“澳人治澳”、高度自治的方针；
2. 必须依法治港治澳，落实中央对特别行政区全面管治权；
3. 落实“爱国者治港”“爱国者治澳”原则。

### 民族平等的表现?
1. 我国各民族具有平等的社会地位；
2. 我国各民族在各个领域享有平等的权利，承担相同的法定义务。

### 为什么要加强巩固民族团结? (为什么要像石榴籽一样紧紧抱在一起?)
1. 加强和巩固民族团结，维护国家统一，是中华民族的最高利益。
2. 团结友爱是我国各民族的宝贵传统。
3. 各族人民只有铸牢中华民族共同体意识，才能追求共同发展、共同富裕和共同繁荣。
4. 维护和促进民族团结，是每个公民的神圣职责和光荣义务。

### 为什么要加快民族地区经济社会文化发展?
1. 逐步缩小发展差距；
2. 促进民族团结和共同繁荣；
3. 发展社会主义民族关系。

### 怎样维护国家统一、反对分裂?
1. 反对分裂，就要维护国家统一、国家主权和领土完整。
2. 反对分裂，就要反对一切形式的民族分裂活动，尤其要坚决反对暴力恐怖活动。
3. 反对分裂，就要维护国家安全。

## 熟读知识点:
1. 全体中华儿女的共同愿望，中华民族根本利益所在，实现中华民族伟大复兴的必然要求是：**解决台湾问题、实现祖国完全统一**。
2. 我国的民族概况：**是统一的多民族国家 (五十六个民族)**。
3. 我国的民族分布格局：**大杂居、小聚居、交错居住**。
4. 国家安全是安邦定国的重要基石。国泰民安是人民群众最基本、最普遍的愿望。

### 怎样加快民族地区经济社会文化发展?
1. 党和国家在人力、物力、财力等方面大力支持。 (如：推动西部大开发战略，实施兴边富民行动)
2. 保障和改善民生。 (如：发展教育，实施积极的就业政策，健全基本养老医保制度)
3. 大力扶持民族文化的保护、继承、创新、和发展，促进民族间文化交流。

### 为什么要维护国家统一，反对分裂?
1. 加强和巩固民族团结，维护国家统一是中华民族的最高利益。
2. 分裂会导致社会动荡、经济发展停滞不前，各族人民就会遭殃。
3. 一切民族分裂行为都将受到法律的制裁。
4. 反对分裂，是爱国主义精神的具体体现，是每个公民义不容辞的责任。

### 青少年为国家安全，祖国统一做些什么?
1. 树立远大理想，热爱祖国；
2. 勤奋学习，努力掌握建设祖国和保卫祖国的本领；
3. 增强国家安全意识，提高防范能力，树立国家利益高于一切的观念；
4. 坚持一个中国原则，履行维护国家安全和国家统一的法定义务；
5. 坚决同一切分裂祖国、破坏祖国统一的言行作斗争。
