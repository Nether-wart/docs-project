# 第二课 创新驱动发展
## 必背知识点:
1. 建设科技创新强国要落实的战略：**科教兴国战略、人才强国战略、创新驱动发展战略**。
2. 综合国力竞争的决定性因素是：**科技创新能力**。
3. 全面创新的核心是 **科技创新**。
4. 创新的目的是 **增进人类福祉，让生活更美好**。
5. 推动创新创造的生力军是 **企业**。提升创新能力是企业持续发展之基、市场制胜之道。
6. **教育**是民族振兴、社会进步的基石。
7. **教育**是提高国民素质、培养创新型人才、促进人的全面发展的根本途径。
8. **自力更生**是中华民族自立于世界民族之林的奋斗基点。
9. **自主创新**是我们攀登世界科技高峰的必由之路。
10. **关键核心技术**是国之重器。

### 我国为什么要实施创新驱动发展战略? (创新对民族国家的重要性?) 
1. 创新是引领发展的第一动力。
2. 创新是推动人类社会向前发展的重要力量。
3. 创新已经成为世界主要国家发展战略的重心。
4. 创新驱动是国家命运所系。
5. 创新是改革开放的生命。改革创新推动中国走向富强。
6. 创新是一个民族进步的灵魂，是一个国家兴旺发达的不竭源泉，也是中华民族最鲜明的民族禀赋。

### 为什么要建设创新强国? (我国的科技发展现状是怎样的?) 
1. **科技创新能力**已经成为综合国力竞争的决定性因素。
2. 我国科技发展的现状：
    1. 十八大以来，我国加快推进科技自立自强，一些关键核心技术实现突破，战略性新兴产业发展壮大，成功进入创新型国家行列。
    2. 我国推进高质量发展还有许多卡点，科技创新能力还需进一步提高。要建成世界科技创新强国，任重道远。

### 怎样建成科技创新强国?
1. 坚持科技是第一生产力、人才是第一资源、创新是第一动力。
2. 实施科教兴国战略、人才强国战略、创新驱动发展战略。
3. 坚持教育优先发展、科技自立自强、人才引领驱动，加快建设教育强国、科技强国、人才强国。
4. 要增强自主创新能力，坚定不移地走中国特色自主创新道路。
5. 加快形成有利于创新的治理格局、活动平台，融资平台，舆论氛围和法治环境。
6. 全面实施素质教育，提高国民素质，培养创新型人才。

### 青少年学生怎样培养创新意识和创新能力?
1. 树立理想，认真学习，勇于担责，立志成才；
2. 提高创新能力，培养创新精神。
3. 注重实践，参加小发明、小制作活动。

## 熟读知识点:
1. 生活的各个领域都需要创新，包括**知识的创新、技术的创新、制度的创新**。
### 创新精神的表现?
1. 敢为人先、敢于冒险的勇气和自信；
2. 探索新知的好奇心和挑战权威的批判精神；
3. 承受挫折的坚强意志和沟通合作的团队精神；
4. 舍我其谁的责任担当和造福人类的济世情怀。

### 为什么要大力发展教育事业?
1. **教育**是民族振兴、社会进步的基石。
2. **教育**是提高国民素质、培养创新型人才、促进人的全面发展的根本途径。
3. **教育**寄托着亿万家庭对美好生活的期盼。
4. 百年大计，**教育为本**。

### 如何正确对待知识产权?
1. 既要尊重他人的知识产权，又要学会保护自己的知识产权。
2. 保护知识产权就是尊重创造、保护创新。
3. 当知识产权受到侵犯时，我们要善于运用法律维护自己的权益。
