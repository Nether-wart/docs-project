# 第六课 建设美丽中国
## 必背知识点:
1. 面对我国严峻的人口、资源、环境形势，我国实行的基本国策分别是：**计划生育、节约资源、保护环境**。
2. 建设生态文明，要坚持**创新、协调、绿色、开放、共享**的新发展理念。
3. 面对突出的人口、资源、环境问题，我们的必然选择是什么? (我们要走什么样的发展道路?) 
	- **转变发展方式，坚持绿色发展，走生产发展、生活富裕、生态良好的文明发展道路**。

### 美丽中国的时代图景?
走绿色发展道路，建设资源节约型，环境友好型社会，实现经济繁荣、生态良好、人民幸福，是建设美丽中国的时代图景。

### 怎样建设美丽中国? (怎样建设生态文明，实现人与自然和谐共生?)  (如何坚持绿色发展道路?)  (怎样使家乡的天更蓝、水更清、山更绿?) 
1. 要坚持**节约资源**和**保护环境**的基本国策。
2. 要坚持**创新、协调、绿色、开放、共享**的新发展理念。
3. 走**生产发展、生活富裕、生态良好**的文明发展道路。
4. 建设资源节约型，环境友好型社会。
5. **走绿色发展之路**，坚持绿色富国，绿色惠民。
6. 建设生态文明，追求人与自然和谐共生，实现**可持续发展**。
7. 处理好**经济发展**与**生态环境保护**的关系。
8. 严守**生态保护红线**、**环境质量底线**、**资源利用上线**。
9. 走**绿色、循环、低碳**发展之路，坚持**节约优先、保护优先、自然恢复为主**的方针。

### 作为中学生 (公民) ，可以为建设美丽家园作出哪些努力?
1. 树立节约资源、保护环境的意识和社会责任感。
2. 积极参加环保活动，为环保建言献策，同破坏环境、浪费资源的行为作斗争。
3. 从身边小事做起，过低碳生活，绿色出行，不使用一次性产品，节约粮食，节水节电，垃圾分类等。 (低碳微行动)

## 熟读知识点:
1. 制约发展的重要因素是 **人口、资源、环境问题**。
2. 人口是社会发展的主体，也是影响经济可持续发展的关键变量。
3. **人口问题**始终是我国面临的全局性、长期性、战略性问题。
4. 自然资源是人类赖以生存的物质基础。人与自然是生命共同体。
5. 当代中国的发展共识：走绿色发展道路，建设生态文明，实现可持续发展。
6. 建设生态文明，要以资源环境承载力为基础，以自然规律为准则，以可持续发展、人与自然和谐共生为目标。

### 经济发展与生态环境之间的关系：
1. 保护生态环境就是保护生产力，改善生态环境就是发展生产力，决不能以牺牲生态环境为代价换取一时的经济增长。
2. 我们既要绿水青山也要金山银山。绿水青山就是金山银山。如果金山银山要以失去绿水青山为代价，那么宁要绿水青山，不要金山银山。
