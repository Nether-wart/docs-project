# 归类知识点背诵

## 一、制度、原则
1.  我国的根本制度：**社会主义制度**。
2.  我国的根本政治制度：**人民代表大会制度**。
3.  我国的基本政治制度：
    1. 中国共产党领导的多党合作和政治协商制度（政党制度）。
    2. 民族区域自治制度。（基本民族政策）
    3. 基层群众自治制度。
4.  我国的基本经济制度：
    1. 公有制为主体、多种所有制经济共同发展。
    2. 按劳分配为主体、多种分配方式并存。（分配制度）
    3. 社会主义市场经济体制。
5.  社会主义经济制度的基础：**生产资料的社会主义公有制**。

---

## 二、方针、原则
1.  解决台湾问题的基本方针（最佳方式）：**“和平统一、一国两制”**。
2.  **一个中国原则**是两岸关系的政治基础。
3.  我国处理民族关系的基本方针：**民族平等、民族团结和各民族共同繁荣**。
4.  我国多党合作的基本方针：**长期共存、互相监督、肝胆相照、荣辱与共**。
5.  我国宪法的基本原则：
    1. 国家的一切权力属于人民
    2. 尊重和保障人权

---

## 三、国策
1. 计划生育（人口）
2. 节约资源（资源）
3. 保护环境（环境）

## 四、战略
1.  建设世界科技创新强国必须实施的战略：
    1. 科教兴国战略
	2. 人才强国战略
	3. 创新驱动发展战略
2.  建设美丽中国要坚持：**可持续发展战略**。
3.  在对外开放中我国奉行的战略：走和平发展道路，奉行互利共赢的开放战略。
4.  在区域协调、城乡融合发展中，要坚持**乡村振兴战略**。

## 五、理念、思想、意识
1.  新发展理念：坚持**创新、协调、绿色、开放、共享**。
2.  我国为应对全球性问题而倡议的理念：**构建人类命运共同体**。
3.  我国谋求自身发展的同时，一直坚持**合作共赢的理念**，分享发展机遇，共享发展成果。
4.  在推动世界格局发展过程中，中国秉持的全球治理观是：**共商共建共享**。
5.  在推动世界格局发展过程中，中国秉持的思想是：**“和而不同”**。
6.  党和政府坚持**以人民为中心的发展思想**。
7.  当代中国的马克思主义、二十一世纪马克思主义：**习近平新时代中国特色社会主义思想**。
8.  “同心掬得满庭芳”，各族人民要铸牢**中华民族共同体意识**。
9.  中国特色社会主义伟大事业的指导思想：
	- 马克思列宁主义
	- 毛泽东思想
	- 邓小平理论
	- “三个代表”重要思想
	- 科学发展观
	- 习近平新时代中国特色社会主义思想

## 六、新
1.  新发展理念：**创新、协调、绿色、开放、共享**。
2.  新型国际关系：**相互尊重、公平正义、合作共赢**。
3.  新发展格局：**国内大循环为主体、国内国际双循环相互促进**。
4.  新时代：**十八大以来（2012年）**，中国特色社会主义进入了新时代，这是我国发展新的历史方位。
5.  新征程：**全面建成小康社会**（2020年），实现第一个百年奋斗目标后，迈上了**全面建设社会主义现代化国家**新征程。
6.  新常态：**经济已由高速增长阶段转向高质量发展阶段**。

## 七、特征、特点、特色、实质
1.  法律的特征：
    1. 法律是由国家制定或认可的。
    2. 法律是由国家强制力保证实施的。（法律最主要特征）
    3. 法律对全体社会成员具有普遍约束力。
2.  犯罪的基本特征：
    1. 严重社会危害性（犯罪最本质的特征）；
    2. 刑事违法性（犯罪的法律标志）
    3. 应受刑罚处罚性（犯罪的必然法律后果）。
3.  中华文化的特点：**源远流长、博大精深、薪火相传、历久弥新**。
4.  当今世界的特点：**开放的、发展的、紧密联系的世界**。
5.  经济全球化的特点（表现）：**商品生产全球化，商品贸易全球化**。
6.  国际竞争的实质：**以经济和科技实力为基础的综合国力的较量**。
7.  当代中国最鲜明的特色：**改革开放**。
8.  中国特色社会主义最本质的特征，中国特色社会主义制度的最大优势是**中国共产党的领导**。
9.  公民基本权利的特点：**广泛性**；公民基本义务的特点：**强制性**。
10. 中华民族精神的特点：**与时俱进**。
11. 网上交往的特点：**虚拟性、平等性、自主性**。
12. 生命的特点：
    1. 生命来之不易
    2. 生命是独特的
    3. 生命是不可逆的
    4. 生命是短暂的

## 十二、中心、目的、目标、决定性因素、要求
1.  社会主义初级阶段所有工作的中心：**以经济建设为中心**。（一个中心）
2.  发展的根本目的：**增进民生福祉**。
3.  创新的目的：**增进人类福祉，让生活更美好**。
4.  科技创新能力已成为综合国力竞争的**决定性因素**。
5.  社会主义的本质要求：**解放和发展社会生产力**。
6.  中国特色社会主义的本质要求：**共同富裕**。
7.  当代中国爱国主义的本质是：**爱国爱党爱社会主义的高度统一**。
8.  两个阶段目标：
    1. 从2020到2035年，在全面建成小康社会的基础上，**基本实现社会主义现代化**。
    2. 从2035到本世纪中叶，把我国建成**富强民主文明和谐美丽的社会主义现代化强国**。
9.  “两个一百年”奋斗目标：
    - 到建党一百年时（1921-2020年），全面建成小康社会（已经实现）；
    - 到建国一百年时（1949-本世纪中叶），全面建成社会主义现代化强国（正在迈进）。
